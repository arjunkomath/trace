Trace

Lightning-fast app launcher for macOS

A native macOS application launcher that runs in your background. Press ⌥Space, type what you need, and launch instantly.

Download GitHub

Features

  • Fuzzy search across all applications
  • Customizable global hotkey (default: ⌥Space)
  • Native macOS design with system theme support
  • Invisible background operation
  • Google search fallback
  • Custom hotkeys per app

Installation

Requirements: macOS 15.5 or later

git clone https://github.com/arjunkomath/trace.git cd trace xcodebuild -project trace.xcodeproj -scheme trace -configuration Release build

Grant permissions when prompted: Accessibility and Apple Events.

Usage

  1. Launch Trace (runs silently in background)
  2. Press ⌥Space to open search
  3. Type to search, press Return to launch
  4. Search "settings" to customize
